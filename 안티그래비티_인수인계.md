## 7. 실제 배포 전에 반드시 교체해야 하는 값

- `window.LANDING_CONFIG.gaMeasurementId`

- `appsScriptUrl: "YOUR_GOOGLE_APPS_SCRIPT_WEBAPP_URL"`
- `gaMeasurementId: "G-XXXXXXXXXX"`

### 7-2. google_apps_script.gs

[google_apps_script.gs](C:/vibe/land/google_apps_script.gs) 안의 아래 값은 반드시 실제 값으로 교체해야 한다.

8. [index.html](C:/vibe/land/index.html) 의 `window.LANDING_CONFIG.appsScriptUrl` 교체
9. GA4 측정 ID 교체
10. 로컬 또는 실제 서버에서 테스트

## 18. 다른 AI에게 바로 전달할 수 있는 요약 프롬프트

아래 문장을 그대로 전달하면 다른 AI가 빠르게 상황을 이해하기 쉽다.

```text
이 프로젝트는 C:\vibe\land 폴더의 브레인시티 메디스파크 분양 랜딩페이지입니다.

현재 상업용 랜딩페이지는 완성되어 있지만 실제 Apps Script URL, Google Sheets ID, GA4 측정 ID는 아직 미입력 상태입니다.
고객 DB에는 이름, 연락처, 개발자정보(이신우)가 숨김값으로 저장되며, 화면에는 개발자명이 노출되지 않게 처리되어 있습니다.
남은 핵심 작업은 Apps Script 실연동, 실기기 테스트, 평면 이미지 실자료 확정 반영입니다.
```

## 19. 최종 인수인계 메모

이 프로젝트는 현재 `기획 완료 + 상업용 프론트 1차 완성 + Apps Script 코드 작성 완료` 상태다.

즉, 완전히 빈 프로젝트는 아니고, 실제 운영 직전 단계에 가깝다.

새 담당자 또는 다른 AI는 새로 기획할 필요 없이 아래 3가지만 우선 처리하면 된다.
