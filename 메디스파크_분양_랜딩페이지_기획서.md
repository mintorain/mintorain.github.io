## 1-1. 참조 페이지 벤치마킹 요약

참조 페이지의 강점은 복잡한 설명보다 즉시 문의 전환에 집중한 단순 구조에 있었다.

- 상단 첫 화면에 단지명, 간단한 사업개요, 전화 및 방문예약 버튼 배치
- 이미지 중심 구성으로 스크롤 피로도 최소화
- 평면 타입을 탭처럼 빠르게 넘겨보는 구조
- `E모델하우스` 구간으로 체류시간 확보
- 하단 관심고객 등록 폼으로 자연스럽게 전환
- 모바일에서 전화 문의와 방문예약 행동을 매우 쉽게 만든 구조

반면 보완할 수 있는 부분도 분명하다.

- 입지 우위와 미래가치에 대한 설득 문장이 약함
- 숫자 기반 신뢰 장치와 차별화 포인트가 충분히 강조되지 않음
- 단순 문의 유도는 강하지만 "왜 이곳이어야 하는가"에 대한 감정적 설득이 얕음

따라서 본 기획서는 참조 페이지의 `간결한 전환 구조`는 유지하되, 아래 3가지를 더 강화하는 방향으로 설계한다.

- 첫 화면 메시지 설득력 강화
- 입지 및 미래가치에 대한 신뢰 기반 카피 강화
- 전화와 폼 전환을 동시에 잡는 이중 CTA 구조 강화

## 2. 마케팅 관점 핵심 인사이트

### 2-1. 이 상품의 가장 강한 한 줄

`병원, 학교, 공원, 상업시설을 모두 도보권으로 누리는 브레인시티 메디컬 라이프의 중심`

이 프로젝트는 일반적인 신도시 아파트처럼 단순히 "새 아파트"를 파는 방식보다, 아래 4개 가치를 묶어 설득하는 방식이 훨씬 유효하다.

- 미래가치: 메디스파크, 아주대병원, 교통 개발 호재
- 실거주가치: 초중학교, 공원, 상업시설 도보권
- 상품가치: 대단지, 남향 위주 배치, 커뮤니티, 수납 특화 평면
- 상징가치: 브레인시티 내 의료·교육·생활 중심축 선점

즉, 랜딩페이지는 "좋은 아파트"보다 "앞으로 더 좋아질 입지의 중심을 먼저 잡는 선택"으로 포지셔닝해야 한다.

## 6. 마케팅 설득 구조

랜딩페이지는 아래 순서로 설계하는 것이 가장 전환율에 유리하다.

### 6-1. Hook

처음 3초 안에 "이 단지는 브레인시티 안에서도 다르다"는 인식을 줘야 한다.

추천 문구:

- 브레인시티에서 병원, 학교, 공원, 상업을 모두 도보로 누리는 중심 입지
- 아주대병원(예정) 바로 앞, 메디컬 라이프의 시작

### 6-2. Trust

막연한 홍보 느낌보다 수치와 사실로 신뢰를 만든다.

- 총 1,215세대 대단지
- 최대 129.1m 동간거리
- 약 1만4천여 평 자연녹지
- 초중학교 도보 1분
- 아주대병원 도보 3분
- 중심상업지역 도보 5분

### 6-3. Desire

타깃이 "살고 싶은 이유"를 구체적으로 상상하게 만든다.

- 아이는 안전하게 걸어서 통학
- 가족은 공원을 집 앞 마당처럼 이용
- 부모는 병원, 상업시설, 교통 편의를 가까이 누림
- 집 안에서는 알파룸, 팬트리, 드레스룸으로 실용적인 생활

### 6-4. Action

페이지 중간과 하단에서 반복적으로 문의를 받는다.

- 상단 고정 전화 버튼 `010-6689-2348`
- 상단 방문예약 버튼
- 중간 배너형 상담 유도
- 하단 폼형 DB 수집
- 모바일 하단 고정 전화 연결 버튼 `010-6689-2348`
- 모바일 하단 고정 방문예약 버튼

상담 버튼의 전환 방식은 아래처럼 설계한다.

- `상담하기` 클릭
- 이름 + 연락처만 입력하는 초간단 레이어 팝업 노출
- 입력 완료 후 `전화 연결하기` 버튼 활성화
- 버튼 클릭 시 `010-6689-2348`로 바로 전화 연결
- 동시에 관리자 측에는 상담 접수 정보가 저장되어 후속 응대 가능하도록 구성

DB 저장 로직 기준:

- 사용자가 `상담하기` 또는 `방문예약` 관련 폼을 제출하면
- Apps Script 웹앱 엔드포인트로 데이터 전송
- Google Sheets에 한 줄씩 누적 저장
- 저장 완료 후 전화 연결 또는 완료 안내 노출
